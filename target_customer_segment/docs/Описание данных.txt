PECOM_МФТИ_TASK2_DS_feedback.csv
	Данные представляют реакцию контрагента на отправляемую нами рассылку. Рассылка - группа маркетинговых компаний проводимых с 1-го по 31-е декабря, 2023 годя
	Здесь.
	UID - уникальный идетнификатор контрагента ПЭКа. Однозанчно характеризует Юридическое, физическое лица или ИП.
					Следует помнить, что контрагент - это участник договора транспортной экспедиции. "Тот, кто возит грузы".
					А рассылка происходит на email, т.е. контакту контрагента. У одного контрагента, особенно у крупных юридических лиц, может
					быть много контактов. И профиль (PECOM_МФТИ_TASK2_DS_behavior.csv) и реакция привязаны к контрагенту. Тогад как конретное  
					действие выполнено контактом.
	nClick		- Клик. Контакт прочитал письмо и нажал на ссылку в нем
	nOpen		- Контакт открыл письмо, вероятно его прочитал. Никаких действий не предпринимал
	nSend		- Контакт не открывал оправленное ему письмо
	
PECOM_МФТИ_TASK2_DS_behavior.csv
	Данные представляют собой описание поведения контрагента ПЭКа за 2023 год
	Здесь.
	UID - уникальный идетнификатор контрагента ПЭКа. Соответствует значению UID из файла PECOM_МФТИ_TASK2_DS_feedback.csv
	UnitType - Тип контрагента. Юридическое лицо, Индивидуальный предприниматель или Физическое лицо
	OKVED	- Общероссийский классификатор ведения экономической деятельности. Код, определяющий вид деятельнрости юридического лица или ИП. Для физиков не определен
	Action - действие контрагента, как-то его характеризующее.
			'Отправляет из' - город отправителя груза данного контрагента
			'Отправляет получателям в ' - город получателя груза контрагента данного контрагента
			'Получает из' - город отправителя груза контрагента данного контрагента
			'Получает в' - город получателя груза данного контрагента
			'Платит за услуги' - наименование услуги, за которую платит контрагент. Список услуг (поле Caption) обширен. Набор характерных услуг - яркий элемент профиля контрагента.
			'Обращание в поддержку' - обращение в поддержку с претензиями (потеряли, сломали, поцарапали) или за консультацией (выдать справку, выписать доверенность, рассказать режим работы отделения)
	Caption - 
		для Action 'Отправляет из', 'Отправляет получателям в ' , 'Получает из',  'Получает в' - наименования городов (филиалов)
		для Action 'Платит за услуги' - наименование услуги, за которую платит контрагент.
		для Action 'Обращание в поддержку'  - "Претензия" или Консультация и обслуживание
		
	BeginDate - дата первого событие из Action-Caption в течении 2023 года
	EndDate - дата последнего события из Action-Caption в течении 2023 года
	nCount - общее количество событий из Action-Caption в течении 2023 года
	Amount - 
		для Action 'Отправляет из', 'Отправляет получателям в ' , 'Получает из',  'Получает в' - средняя объявленная стоимость груза. 
		для Action 'Платит за услуги' - средняя сумма оплаты по данной услуге
		для Action 'Обращание в поддержку'  - среднее время в сутках, на выполнение обращения. (0 - решили день в день, более - число дней)		
	
	Volume - 
		для Action 'Отправляет из', 'Отправляет получателям в ' , 'Получает из',  'Получает в' - средний объем груза в метрах кубических по направлению
		для Action 'Платит за услуги' - не заполняется (NULL)
		для Action 'Обращание в поддержку'  - не заполняется (NULL)		

	Weight - 
		для Action 'Отправляет из', 'Отправляет получателям в ' , 'Получает из',  'Получает в' - средний вес груза в килограммах по направлению
		для Action 'Платит за услуги' - не заполняется (NULL)
		для Action 'Обращание в поддержку'  - не заполняется (NULL)		

PECOM_МФТИ_TASK2_DS_behavior.csv
	Данные представляют собой описание поведения контрагента ПЭКа за 2024 год. С января по март, включительно. Предполагается, что рассылка могла изменить его поведение.
	Внимание! Следует учитывать сесонность перевозок. Данные за первый квартал могут быть не репрезентативны. Файл приведен как дополнительнрый справочный. 
	Не исключено, что их не стоит учитывать.
	Здесь.
	Структура идентична PECOM_МФТИ_TASK2_DS_behavior.csv
	
Примечание. В ходе брифинга, было высказано пожелание получить данные о кликах контактов по каждой акции (по каждой рассылке). Я предварительнро пообщеал, но 
			по здравому размышлению не хочу этого делать. Я не хотел бы в качестве результата получить отношение контакта контрагента к той или иной рассылке.
